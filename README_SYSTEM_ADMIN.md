# システム管理者機能 - クイックリファレンス

## 🚀 クイックスタート

### 1. データベースセットアップ

SupabaseのSQL Editorで以下を実行：

```sql
-- supabase/migration_add_system_admin.sql の内容を実行
```

### 2. システム管理者でログイン

- **URL**: `/system-admin/login`
- **ユーザー名**: `admin`
- **パスワード**: `admin123`

### 3. 店舗を作成

1. 「店舗管理」をクリック
2. 「新規店舗追加」をクリック
3. 店舗情報を入力して作成

## 📋 機能一覧

### システム管理者
- ✅ 店舗の作成・編集・削除
- ✅ 店舗一覧の表示
- 🔜 システム管理者の管理（今後実装）
- 🔜 利用統計（今後実装）

### 店舗管理者
- ✅ メニューの投稿・編集・削除
- ✅ 週間・月間メニューの設定
- ✅ プロフィール編集

### 利用者
- ✅ 店舗IDを入力してメニューを閲覧
